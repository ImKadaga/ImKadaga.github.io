Description

The Challenge is a 29-page illustrated (short) story of Paul Morphy and Howard Staunton, consisting mainly of their correspondence. It is a fan project created in Inkscape, and it's free to download in multiple formats. You can share this work, but it has to remain free for everyone!

Download

  • PDF (A4)
  • CBZ (A4@150 dpi): Comic book format for smaller tablets screens
  • CBZ (A4@300 dpi): Comic book format for larger (FHD+) tablets and screens
  • SVG (source files)

Extras

  • The Illustrated London News - Aug. 28, 1858 (page 1) (PDF)
  • The Illustrated London News - Aug. 28, 1858 (page 11: chess column) (PDF)
  • The Chess Monthly - July, 1858 (full, 57 pages) (PDF)

Description

An unofficial, modified version of Kanazawa Shogi Lite. The original version is free, so this is free, too! This is only a mod for Shogi fans. I don’t maintain this app anymore (last updated: 2020-03-21), I list this as an “archived” project.

  • The first one-kanji set is replaced by the “kanji light 3D OTB” set (v1.1)
  • Internationalized set is included (this is the default)
  • Doubutsu piecet set is included (see: Wikipedia: Dōbutsu shōgi)
  • Kanji Red Wood piece set is included (you have to win a game against the computer to unlock it)
  • Main menu and some graphics are changed
  • The app icon is changed
  • Some words/string are changed in the menus

Download APK

Extras:

  • Shogi moves cheatsheet A4L PDF v2
  • Shogi moves cheatsheet A4L PDF v3
  • Hungarian version of the Shogi cheatsheet (Sógi súgólap)
