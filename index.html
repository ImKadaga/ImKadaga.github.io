I'm a freelance graphic designer, digital artist, and open source software contributor.

About my work

  • I make graphic designs (logo, website, UI, icon, etc.).

Info

An unofficial, modified version of Kanazawa Shogi Lite for Shogi fans. I don’t maintain this app anymore (last update: 2020-03-21), I list this as an “archived” project.

  • The first one-kanji set is replaced by the “kanji light 3D OTB” set (v1.1)
  • Internationalized set is included (this is the default)
  • Doubutsu piecet set is included (see: Wikipedia: Dōbutsu shōgi)
  • Kanji Red Wood piece set is included (you have to win a game against the computer to unlock it)
  • Main menu and some graphics are changed
  • The app icon is changed
  • Some words/string are changed in the menus

Download APK

Extras:

  • Shogi moves cheatsheet A4L PDF v2
  • Shogi moves cheatsheet A4L PDF v3
  • Hungarian version of the Shogi cheatsheet (Sógi súgólap)
