Shogi

Kanazawa Shogi Lite Ka Mod v1.1

An unofficial, modified version of Kanazawa Shogi Lite. The original version is free, so this is also free! No money involved, this is only a mod by a Shogi fan to help spreading Shogi.

  • The first one-kanji set is replaced by the "kanji light 3D OTB" set (v1.1)
  • Internationalized set is included (this is the default)
  • Doubutsu piecet set is included (see: https://en.wikipedia.org/wiki/D%C5%8Dbutsu_sh%C5%8Dgi#Variation)
  • Kanji Red Wood piece set is included (you have to win a game against the computer to unlock it)
  • Main menu and some graphics were changed
  • Got a new app icon
  • Some words/string were changed in the menus

Download: https://www59.zippyshare.com/v/3q9EXTIF/file.html or https://dropapk.to/s8fsb0gzbu5b/Kanazawa_Shogi_Lite_(Ka_mod_v1.1).apk
